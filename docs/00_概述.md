# 智能训练平台 — 后端 API 文档

## 1. 文档说明

本文档为**智能训练平台（Train Platform）** 后端服务的外部调用 API 参考手册，适用于需要通过 HTTP/WebSocket 接口集成本平台能力的第三方系统或前端应用。

文档按功能模块拆分为多个文件，请按需查阅。

## 2. 基础信息

| 项目 | 说明 |
|------|------|
| **Base URL** | `http://<host>:<port>/api/v2` |
| **协议** | HTTP/1.1 + WebSocket |
| **数据格式** | JSON（`Content-Type: application/json`） |
| **字符编码** | UTF-8 |
| **API 版本** | v2 |
| **框架** | FastAPI (Python) |

## 3. 通用约定

### 3.1 请求方式

| 方法 | 语义 |
|------|------|
| `GET` | 查询/列表 |
| `POST` | 创建/触发动作 |
| `PATCH` | 局部更新 |
| `PUT` | 整体更新/替换 |
| `DELETE` | 删除 |

### 3.2 通用分页参数

大部分列表接口支持分页，统一使用以下 Query 参数：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `page` | int | 1 | 页码，最小值 1 |
| `page_size` | int | 50 | 每页条数，范围 1~500 |

分页响应统一结构：

```json
{
  "items": [...],
  "meta": {
    "page": 1,
    "page_size": 50,
    "total": 120
  }
}
```

### 3.3 通用响应格式

#### 成功响应

HTTP 状态码：`200`（查询/更新）、`201`（创建）、`202`（异步接受）

响应体为对应的资源对象或列表。

#### 删除响应

```json
{
  "ok": true,
  "message": "Resource deleted"
}
```

#### 错误响应

| HTTP 状态码 | 说明 |
|-------------|------|
| 400 | 参数校验错误 (ValidationError) |
| 404 | 资源未找到 (NotFoundError) |
| 409 | 资源冲突 (ConflictError) |
| 500 | 服务端内部错误 |

错误响应体：

```json
{
  "message": "Validation error",
  "detail": "具体错误描述"
}
```

### 3.4 时间格式

所有时间字段均为 **ISO 8601** 格式，示例：`2026-03-30T01:36:28+00:00`

### 3.5 枚举值一览

#### DatasetType / TaskType
- `detection` — 目标检测
- `segmentation` — 语义分割
- `classification` — 图像分类

#### TrainingRunStatus
- `created` — 已创建
- `queued` — 已排队
- `running` — 训练中
- `completed` — 已完成
- `failed` — 已失败
- `cancelled` — 已取消
- `deleted` — 已删除

#### DeploymentStatus
- `pending` — 待部署
- `deploying` — 部署中
- `active` — 已激活
- `inactive` — 未激活
- `failed` — 失败
- `deleting` — 删除中

#### DeploymentPlatform
- `local` — 本地部署
- `docker` — Docker 部署
- `kubernetes` — Kubernetes 部署
- `aws` / `azure` / `gcp` — 云平台部署

#### ModelStage
- `development` — 开发中
- `testing` — 测试中
- `production` — 生产环境
- `deprecated` — 已废弃

#### LogLevel
- `DEBUG` / `INFO` / `WARNING` / `ERROR`

## 4. 静态资源访问

平台通过以下 URL 前缀提供静态文件访问：

| URL 前缀 | 说明 |
|----------|------|
| `/static/datasets/` | 数据集文件 |
| `/static/thumbnails/` | 缩略图 |
| `/static/training/` | 训练产物（权重、日志等） |
| `/static/temp/` | 临时文件（上传、转换产物等） |
| `/static/pretrain/` | 预训练模型 |

## 5. 健康检查

```
GET /health
```

**响应：**
```json
{
  "ok": true
}
```
