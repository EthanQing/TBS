# 数据集转换与增强 API

> **路由前缀：**  
> - 格式转换：`/api/v2/dataset-conversions`（标签：`dataset-conversions`）  
> - 数据增强：`/api/v2/datasets/{dataset_id}/augmentations`（标签：`dataset-augmentations`）

---

## 一、数据集格式转换 (`/api/v2/dataset-conversions`)

将 JSON 标注格式的数据集 ZIP 转换为 YOLO 或 COCO 格式。

### 1. 创建转换任务

```
POST /api/v2/dataset-conversions
```

**Content-Type:** `multipart/form-data`

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file` | File | ✅ | 包含图片和 JSON 标注的 ZIP 文件 |
| `target_format` | string | 否 | 目标格式：`yolo`（默认）/ `coco` |

**响应：** `201 Created`

```json
{
  "job_id": "abc123def456",
  "status": "queued",
  "stage": "queued",
  "progress": 0,
  "processed": 0,
  "total": null,
  "target_format": "yolo",
  "logs": ["Received: dataset.zip", "Target format: yolo"],
  "output_url": null,
  "output_filename": null,
  "error_message": null,
  "created_at": "2026-03-20T10:00:00+00:00",
  "updated_at": "2026-03-20T10:00:00+00:00"
}
```

转换在后台异步执行，可通过轮询或 WebSocket 获取进度。

---

### 2. 查询转换进度

```
GET /api/v2/dataset-conversions/{job_id}
```

**响应：**

```json
{
  "job_id": "abc123def456",
  "status": "running",
  "stage": "converting_labels",
  "progress": 45,
  "processed": 90,
  "total": 200,
  "logs": ["Extracting zip...", "[90/200] image_090.json"],
  "output_url": null,
  "error_message": null
}
```

#### 转换阶段 (stage)

| 阶段 | 说明 |
|------|------|
| `queued` | 排队中 |
| `extracting` | 解压中 |
| `converting_labels` | 转换标注 |
| `copying_images` | 复制图片 |
| `converting_coco` | 转换为 COCO 格式（仅 target_format=coco） |
| `writing_data_yaml` | 生成 data.yaml（仅 target_format=yolo） |
| `zipping` | 打包 ZIP |
| `done` | 完成 |
| `failed` | 失败 |

---

### 3. 下载转换结果

```
GET /api/v2/dataset-conversions/{job_id}/download
```

任务完成后（`status=completed`），可通过此接口下载转换后的 ZIP 文件。

**响应：** `application/zip` 文件流

---

## 二、数据增强 (`/api/v2/datasets/{dataset_id}/augmentations`)

对数据集应用数据增强变换，生成增强后的新数据。

### 1. 预览增强效果

```
POST /api/v2/datasets/{dataset_id}/augmentations/preview
```

在不实际修改数据集的情况下，预览增强变换效果。

**请求体：** `DatasetAugmentationPreviewRequest`

**响应：** 预览结果包含原始和增强后的图片对比。

---

### 2. 创建增强任务

```
POST /api/v2/datasets/{dataset_id}/augmentations
```

**响应：** `201 Created`，返回 `DatasetAugmentationJobOut`。

增强任务在后台异步执行。

---

### 3. 查询增强任务

```
GET /api/v2/datasets/{dataset_id}/augmentations/{job_id}
```

---

### 4. 取消增强任务

```
POST /api/v2/datasets/{dataset_id}/augmentations/{job_id}/cancel
```

**响应：**

```json
{
  "job_id": "aug-001",
  "status": "cancelled",
  "cancel_requested": true
}
```

---

### 5. 发布增强结果

```
POST /api/v2/datasets/{dataset_id}/augmentations/{job_id}/publish
```

将增强完成的数据合并到数据集中，创建新版本。

---

### 6. 增强进度实时流 (WebSocket)

```
ws://<host>/api/v2/datasets/{dataset_id}/augmentations/{job_id}/stream?from_seq=0&from_result_id=0
```

#### 消息类型

| type | 说明 |
|------|------|
| `snapshot` | 初始任务快照 |
| `progress` | 进度更新 |
| `item` | 单张增强结果 |
| `done` | 任务结束 |
| `error` | 错误 |
| `ping` | 心跳（每 15 秒） |

#### progress 消息示例

```json
{
  "type": "progress",
  "data": {
    "job_id": "aug-001",
    "dataset_id": 1,
    "status": "running",
    "phase": "augmenting",
    "progress": 60,
    "processed": 600,
    "total": 1000,
    "seq": 30,
    "last_result_id": 600
  }
}
```

---

## 数据模型

### DatasetConversionOut

| 字段 | 类型 | 说明 |
|------|------|------|
| `job_id` | string | 任务 ID |
| `status` | string | 状态：`queued`/`running`/`completed`/`failed` |
| `stage` | string | 当前阶段 |
| `progress` | int | 进度百分比 (0~100) |
| `processed` | int | 已处理文件数 |
| `total` | int? | 总文件数 |
| `target_format` | string | 目标格式 |
| `logs` | string[] | 日志列表 |
| `output_url` | string? | 下载 URL（完成后） |
| `output_filename` | string? | 输出文件名 |
| `error_message` | string? | 错误信息 |
| `created_at` | string | 创建时间 |
| `updated_at` | string | 更新时间 |

### DatasetAugmentationJobOut

| 字段 | 类型 | 说明 |
|------|------|------|
| `job_id` | string | 任务 ID |
| `dataset_id` | int | 数据集 ID |
| `status` | string | 状态 |
| `phase` | string? | 阶段 |
| `progress` | int | 进度百分比 |
| `processed` | int | 已处理数 |
| `total` | int | 总数 |
| `seq` | int | 序列号 |
| `cancel_requested` | bool | 是否已请求取消 |
| `error_message` | string? | 错误信息 |
| `result` | object? | 结果 |
| `created_at` | datetime? | 创建时间 |
| `updated_at` | datetime? | 更新时间 |
