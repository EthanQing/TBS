# 数据集管理 API (Datasets)

> **路由前缀：** `/api/v2/datasets`  
> **标签：** `datasets`

数据集是训练平台的数据基础，支持 YOLO / COCO 格式，具备版本管理、文件管理、拆分、统计等功能。

---

## 1. 获取数据集列表

```
GET /api/v2/datasets
```

### 请求参数 (Query)

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码 |
| `page_size` | int | 否 | 50 | 每页条数 (1~500) |
| `format` | string | 否 | - | 按格式过滤：`yolo` / `coco` |

### 响应示例

```json
{
  "items": [
    {
      "dataset_id": 1,
      "name": "行人数据集-v1",
      "dataset_type": "detection",
      "format": "yolo",
      "storage_path": "datasets/1",
      "description": "城市场景行人检测数据集",
      "active_version_id": 3,
      "created_at": "2026-03-01T10:00:00+00:00",
      "updated_at": "2026-03-15T08:30:00+00:00",
      "statistics": {
        "num_images": 1200,
        "num_classes": 3,
        "size_mb": 256.5
      }
    }
  ],
  "meta": { "page": 1, "page_size": 50, "total": 1 }
}
```

## 2. 创建数据集

```
POST /api/v2/datasets
```

### 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | string | ✅ | 数据集名称 (1~255 字符) |
| `dataset_type` | string | ✅ | 类型：`detection` / `segmentation` / `classification` |
| `format` | string | 否 | 数据格式，默认 `yolo` |
| `description` | string | 否 | 描述 |

### 请求示例

```json
{
  "name": "行人数据集-v1",
  "dataset_type": "detection",
  "format": "yolo",
  "description": "城市场景行人检测"
}
```

### 响应

- **状态码：** `201 Created`
- **响应体：** `DatasetOut` 对象

## 4. 获取数据集详情（扩展版）

```
GET /api/v2/datasets/{dataset_id}/detail
```

返回数据集 + 统计信息 + 当前激活版本 + 最近版本列表 + 最近事件列表。

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `versions_limit` | int | 20 | 返回的版本数限制 (0~200) |
| `events_limit` | int | 20 | 返回的事件数限制 (0~200) |

### 响应示例

```json
{
  "dataset": { "dataset_id": 1, "name": "...", "..." : "..." },
  "statistics": {
    "dataset_id": 1,
    "version_id": 3,
    "version": 3,
    "total_files": 2400,
    "total_size_bytes": 268435456,
    "total_size_mb": 256.0,
    "total_images": 1200,
    "annotations_count": 5800
  },
  "active_version": { "version_id": 3, "..." : "..." },
  "versions": [],
  "events": []
}
```

## 7. 数据上传

### 7.1 上传归档（覆盖式）

```
POST /api/v2/datasets/{dataset_id}/upload
```

上传 ZIP 归档（YOLO 格式），替换数据集内容。

**Content-Type:** `multipart/form-data`

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file` | File | ✅ | ZIP 归档文件 |
| `message` | string | 否 | 版本说明 |
| `created_by` | string | 否 | 操作人 |
| `create_version` | bool | 否 | 是否创建新版本（默认 true） |
| `activate` | bool | 否 | 自动激活新版本（默认 true） |
| `split_enabled` | bool | 否 | 上传后自动拆分（默认 false） |
| `split_train_ratio` | float | 否 | 训练集比例 |
| `split_val_ratio` | float | 否 | 验证集比例 |
| `split_test_ratio` | float | 否 | 测试集比例 |
| `split_seed` | int | 否 | 随机种子 |
| `split_shuffle` | bool | 否 | 是否打乱 |
| `split_overwrite` | bool | 否 | 是否覆盖已有拆分 |

### 7.2 导入数据集（兼容接口）

```
POST /api/v2/datasets/import
POST /api/v2/datasets/upload
```

与 7.1 类似，但通过 Form 传递 `dataset_id`。需先创建数据集再上传。

### 7.3 追加上传

```
POST /api/v2/datasets/{dataset_id}/append
```

将 ZIP 内容追加到已有数据集。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file` | File | ✅ | ZIP 归档文件 |
| `message` | string | 否 | 版本说明 |
| `created_by` | string | 否 | 操作人 |
| `create_version` | bool | 否 | 默认 true |
| `activate` | bool | 否 | 默认 true |

### 7.4 上传图片与标注

```
POST /api/v2/datasets/{dataset_id}/uploads/images
```

上传单张或多张图片及对应标注文件。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `files` / `images` | File[] | ✅ | 图片文件 |
| `labels` / `annotations` | File[] | 否 | 标注文件（YOLO txt 格式） |
| `relative_dir` | string | 否 | 图片存储子目录，默认 `images` |
| `labels_relative_dir` | string | 否 | 标注存储子目录 |
| `require_labels` | bool | 否 | 是否要求必须有标注（默认 true） |
| `message` | string | 否 | 操作说明 |
| `created_by` | string | 否 | 操作人 |
| `create_version` | bool | 否 | 默认 true |
| `create_snapshot` | bool | 否 | 默认 false |
| `activate` | bool | 否 | 默认 true |

## 8. 版本管理

### 8.1 获取版本列表

```
GET /api/v2/datasets/{dataset_id}/versions
```

分页返回数据集所有版本。

### 8.2 创建版本

```
POST /api/v2/datasets/{dataset_id}/versions
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `message` | string | 否 | 版本说明 |
| `created_by` | string | 否 | 操作人 |
| `create_snapshot` | bool | 否 | 是否创建快照拷贝（默认 false） |

### 8.3 激活版本

```
POST /api/v2/datasets/{dataset_id}/versions/{version_id}/activate
```

将指定版本设为当前活跃版本。

### 8.4 版本差异对比

```
GET /api/v2/datasets/{dataset_id}/versions/{version_id}/diff
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `base_version_id` | int | 否 | 基准版本 ID（默认上一版本） |
| `limit` | int | 否 | 文件列表限制（默认 200） |

**响应示例：**

```json
{
  "dataset_id": 1,
  "base_version_id": 2,
  "base_version": 2,
  "version_id": 3,
  "version": 3,
  "summary": { "added": 50, "removed": 5, "modified": 10 },
  "added": ["images/train/new_001.jpg", "..."],
  "removed": ["images/train/old_001.jpg"],
  "modified": ["labels/train/img_100.txt"],
```

## 11. 数据集拆分

### 11.1 执行拆分

```
POST /api/v2/datasets/{dataset_id}/split
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `version_id` | int | 否 | 目标版本 |
| `train_ratio` | float | 否 | 训练集比例（默认 0.9，范围 0~1） |
| `val_ratio` | float | 否 | 验证集比例 |
| `test_ratio` | float | 否 | 测试集比例 |
| `seed` | int | 否 | 随机种子 |
| `shuffle` | bool | 否 | 是否打乱（默认 true） |
| `overwrite` | bool | 否 | 是否覆盖已有拆分（默认 true） |

**响应示例：**

```json
{
  "dataset_id": 1,
  "version_id": 3,
  "version": 3,
  "total_images": 1200,
  "train_count": 1080,
  "val_count": 120,
  "test_count": 0,
  "train_ratio": 0.9,
  "val_ratio": 0.1,
  "test_ratio": 0.0,
  "seed": 42,
  "shuffle": true
}
```

### 11.2 查询拆分结果

```
GET /api/v2/datasets/{dataset_id}/split
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `version_id` | int | 版本 ID |
| `split` | string | 按拆分类型过滤：`train` / `val` / `test` |

## 16. 违规数据集转换

```
POST /api/v2/datasets/{dataset_id}/convert
```

**状态码：** `202 Accepted`（异步任务）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `label_strategy` | string | ✅ | 标签策略：`full` / `leaf` / `root` / `level` / `mapping` |
| `label_level` | int | 条件必填 | 当 `label_strategy=level` 时必须，≥1 |
| `label_separator` | string | 否 | 标签分隔符（默认 `%`） |
| `label_mapping` | object | 否 | 标签映射表 |
| `slice_size` | int | 否 | 切片大小（像素，64~8192） |
| `overlap` | float | 否 | 切片重叠率（0~1） |
| `padding` | int | 否 | 标注框额外填充 |
| `min_area_ratio` | float | 否 | 最小面积比 |
| `min_visibility` | float | 否 | 最小可见度 |
| `min_pixel_size` | int | 否 | 最小像素尺寸 |
| `negative_ratio` | float | 否 | 负样本比例 |
| `empty_positive_action` | string | 否 | 空正样本处理：`discard` / `negative` |
| `split_enabled` | bool | 否 | 转换后自动拆分 |
| `split_*` | ... | 否 | 拆分参数（同拆分接口） |

**响应：**

```json
{
  "job_id": "abc123",
  "status": "queued"
}
```

### 转换进度 WebSocket

```
ws://<host>/api/v2/datasets/{dataset_id}/convert/{job_id}/stream?from_seq=0
```

消息类型：
- `snapshot` — 初始状态快照
- `progress` — 进度更新
- `done` — 完成
- `error` — 错误
- `ping` — 心跳（每 15 秒）
