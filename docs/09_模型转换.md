# 模型转换 API (Model Conversions)

> **路由前缀：** `/api/v2/model-conversions`  
> **标签：** `model-conversions`

模型转换模块用于将 PyTorch 权重文件（`.pt` / `.pth`）转换为 ONNX 格式，并自动执行性能基准测试。

---

## 1. 创建模型转换任务

```
POST /api/v2/model-conversions
```

**Content-Type:** `multipart/form-data`

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `file` | File | ✅ | - | PyTorch 权重文件 (.pt/.pth) |
| `source_format` | string | 否 | `pt` | 源格式：`pt` / `pth` |
| `target_format` | string | 否 | `onnx` | 目标格式：`onnx` |
| `opset` | int | 否 | null | ONNX opset 版本 |
| `dynamic` | bool | 否 | true | 是否使用动态轴 |

> 当前仅支持 YOLOv8 等 Ultralytics 模型的 PT → ONNX 转换。

### 响应

**状态码：** `201 Created`

```json
{
  "job_id": "abc123def456",
  "status": "queued",
  "progress": 0,
  "logs": ["已接收文件: model.pt", "source_format=pt target_format=onnx"],
  "output_url": null,
  "output_filename": null,
  "error_message": null,
  "performance": null,
  "created_at": "2026-03-20T10:00:00+00:00",
  "updated_at": "2026-03-20T10:00:00+00:00"
}
```

转换任务异步执行，客户端轮询查询进度。

## 2. 查询转换进度

```
GET /api/v2/model-conversions/{job_id}
```

## 转换流程说明

1. **文件上传** — 接收 PT 权重文件保存到临时目录
2. **任务分发** — 将转换任务分发到推理 Worker
3. **模型加载** — Worker 加载 Ultralytics YOLO 模型
4. **ONNX 导出** — 执行 `model.export(format="onnx")` 
5. **性能测试** — 对 PT 和 ONNX 模型分别执行推理基准测试
6. **结果返回** — 提供下载 URL 和性能对比数据

---

## 性能数据说明

| 字段 | 说明 |
|------|------|
| `device` | 测试设备：`cuda` / `cpu` |
| `imgsz` | 测试输入尺寸 |
| `pt.size_mb` | PT 模型文件大小 |
| `pt.latency_ms` | PT 单次推理延迟 |
| `pt.throughput_img_s` | PT 吞吐量（张/秒） |
| `onnx.size_mb` | ONNX 模型文件大小 |
| `onnx.latency_ms` | ONNX 单次推理延迟 |
| `onnx.throughput_img_s` | ONNX 吞吐量（张/秒） |

> 性能测试为 best-effort（尽力执行），不会影响转换结果。测试失败时仍返回模型文件大小。

---

## 数据模型

### ModelConversionOut

| 字段 | 类型 | 说明 |
|------|------|------|
| `job_id` | string | 任务 ID |
| `status` | string | 状态：`queued` / `running` / `completed` / `failed` |
| `progress` | int | 进度百分比 (0~100) |
| `logs` | string[] | 日志行列表 |
| `output_url` | string? | ONNX 文件下载 URL（完成后） |
| `output_filename` | string? | 输出文件名 |
| `error_message` | string? | 错误信息 |
| `performance` | object? | 性能对比数据 |
| `created_at` | string | 创建时间 |
| `updated_at` | string | 更新时间 |
