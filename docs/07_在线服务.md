# 在线服务 API (Serving)

> **路由前缀：** `/api/v2/serving`  
> **标签：** `serving`

在线服务（Serving）模块提供部署后的模型推理 API，供外部系统通过 API Key 鉴权调用。这是面向生产环境的推理接口。

---

## 鉴权方式

所有在线推理请求需要携带部署时生成的 API Key，支持以下两种方式：

### 方式一：自定义请求头

```
X-Deployment-Key: sk-xxxxxxxxxxxx
```

### 方式二：Bearer Token

```
Authorization: Bearer sk-xxxxxxxxxxxx
```

> API Key 在执行部署 (`POST /api/v2/deployments/{id}/execute`) 时生成，仅返回一次。

---

## 1. 健康检查

```
GET /api/v2/serving/deployments/{deployment_id}/health
```

验证部署是否就绪。

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `auth_required` | bool | true | 是否需要鉴权 |

**响应：**

```json
{
  "ok": true,
  "deployment_id": 1,
  "model_version_id": 1,
  "endpoint_url": "http://localhost:8080",
  "status": "active"
}
```

**可能的错误：**

| 状态码 | 说明 |
|--------|------|
| 401 | API Key 缺失或无效 |
| 404 | 部署不存在 |
| 409 | 部署未激活或未配置 API Key |

## 2. 在线推理

```
POST /api/v2/serving/deployments/{deployment_id}/infer
```

支持两种请求格式：

### 2.1 JSON 方式

```
Content-Type: application/json
X-Deployment-Key: sk-xxxxxxxxxxxx
```

**请求体：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `image_url` | string | 二选一 | 图片 HTTP(S) URL |
| `conf` | float | 否 | 置信度阈值（默认 0.5） |
| `iou` | float | 否 | IOU 阈值（默认 0.45） |
| `show_labels` | bool | 否 | 输出显示标签（默认 true） |
| `show_confidence` | bool | 否 | 输出显示置信度（默认 true） |

```json
{
  "image_url": "https://example.com/photo.jpg",
  "conf": 0.5,
  "iou": 0.45
}
```

### 2.2 表单上传方式

```
Content-Type: multipart/form-data
X-Deployment-Key: sk-xxxxxxxxxxxx
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file` | File | 二选一 | 图片文件 |
| `image_url` | string | 二选一 | 图片 URL |
| `conf` | float | 否 | 置信度阈值 |
| `iou` | float | 否 | IOU 阈值 |
| `show_labels` | string | 否 | `true`/`false` |
| `show_confidence` | string | 否 | `true`/`false` |

> ⚠️ `file` 和 `image_url` 必须提供恰好一个。

### 响应示例

```json
{
  "deployment_id": 1,
  "model_version_id": 1,
  "engine": "ultralytics",
  "family": "yolo",
  "variant": "yolov8n",
  "run_id": "abc-123",
  "input_path": "serving_uploads/xyz.jpg",
  "input_meta": {
    "show_labels": true,
    "show_confidence": true
  },
  "inference_time_ms": 12.5,
  "names": {
    "0": "person",
    "1": "car",
    "2": "bicycle"
  },
  "predictions": [
    {
      "class_id": 0,
      "class_name": "person",
      "confidence": 0.92,
      "bbox": [100, 200, 300, 500]
    },
    {
      "class_id": 1,
      "class_name": "car",
      "confidence": 0.87,
      "bbox": [400, 300, 800, 600]
    }
  ],
  "raw_output": {
    "predictions": [...],
    "names": {...}
  }
}
```

## 调用示例

### cURL 示例

```bash
# 使用图片 URL 推理
curl -X POST "http://localhost:8000/api/v2/serving/deployments/1/infer" \
  -H "X-Deployment-Key: sk-xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"image_url": "https://example.com/test.jpg", "conf": 0.5}'

# 上传文件推理
curl -X POST "http://localhost:8000/api/v2/serving/deployments/1/infer" \
  -H "X-Deployment-Key: sk-xxxxxxxxxxxx" \
  -F "file=@/path/to/image.jpg" \
  -F "conf=0.5"
```

### Python 示例

```python
import requests

API_URL = "http://localhost:8000/api/v2/serving/deployments/1/infer"
API_KEY = "sk-xxxxxxxxxxxx"

# 方式一：传入图片 URL
response = requests.post(
    API_URL,
    headers={
        "X-Deployment-Key": API_KEY,
        "Content-Type": "application/json",
    },
    json={
        "image_url": "https://example.com/test.jpg",
        "conf": 0.5,
        "iou": 0.45,
    },
)
result = response.json()
print(f"检测到 {len(result['predictions'])} 个目标")
for pred in result["predictions"]:
    print(f"  {pred['class_name']}: {pred['confidence']:.2f}")

# 方式二：上传本地文件
with open("/path/to/image.jpg", "rb") as f:
    response = requests.post(
        API_URL,
        headers={"X-Deployment-Key": API_KEY},
        files={"file": f},
        data={"conf": "0.5"},
    )
```

## 数据模型

### ServingInferResponse

| 字段 | 类型 | 说明 |
|------|------|------|
| `deployment_id` | int | 部署 ID |
| `model_version_id` | int | 模型版本 ID |
| `engine` | string? | 推理引擎 |
| `family` | string? | 模型族 |
| `variant` | string? | 模型变体 |
| `run_id` | string? | 源训练任务 ID |
| `input_path` | string? | 输入文件路径 |
| `input_meta` | object? | 输入元信息 |
| `inference_time_ms` | float? | 推理耗时（毫秒） |
| `names` | object? | 类名映射（class_id → name） |
| `predictions` | array | 检测结果列表 |
| `raw_output` | object? | 原始输出 |
