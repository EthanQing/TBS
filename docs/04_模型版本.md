# 模型版本 API (Model Versions)

> **路由前缀：** `/api/v2/model-versions`  
> **标签：** `model-versions`

模型版本是训练完成后注册到模型注册表的产物，可用于部署和推理。

---

## 1. 获取模型版本列表

```
GET /api/v2/model-versions
```

### 请求参数 (Query)

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码 |
| `page_size` | int | 否 | 50 | 每页条数 (1~500) |
| `project_id` | int | 否 | - | 按项目过滤 |
| `run_id` | string | 否 | - | 按训练任务 ID 过滤 |
| `stage` | string | 否 | - | 按阶段过滤：`development` / `testing` / `production` / `deprecated` |

### 响应示例

```json
{
  "items": [
    {
      "model_version_id": 1,
      "project_id": 1,
      "run_id": "abc-123",
      "version": "v1.0",
      "stage": "production",
      "description": "首次训练模型",
      "weights_path": "/static/training/abc-123/weights/best.pt",
      "model_size_mb": 12.5,
      "metrics": { "mAP50": 0.85 },
      "created_at": "2026-03-01T12:00:00+00:00",
      "updated_at": "2026-03-01T12:00:00+00:00"
    }
  ],
  "meta": { "page": 1, "page_size": 50, "total": 1 }
}
```

---

## 2. 注册模型版本

```
POST /api/v2/model-versions
```

从已完成的训练任务注册模型版本。

### 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `run_id` | string | ✅ | 训练任务 ID |
| `version` | string | 否 | 版本号（如 `v1.0`） |
| `stage` | string | 否 | 阶段：`development`（默认）/ `testing` / `production` / `deprecated` |
| `description` | string | 否 | 说明 |

### 请求示例

```json
{
  "run_id": "abc-123",
  "version": "v1.0",
  "stage": "production",
  "description": "行人检测模型首次发布"
}
```

### 响应

- **状态码：** `201 Created`
- **响应体：** `ModelVersionOut` 对象

---

## 3. 获取模型版本详情

```
GET /api/v2/model-versions/{model_version_id}
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `model_version_id` | int | 模型版本 ID |

---

## 4. 更新模型版本

```
PATCH /api/v2/model-versions/{model_version_id}
```

可更新版本号、阶段、描述等信息。

---

## 数据模型

### ModelVersionOut

| 字段 | 类型 | 说明 |
|------|------|------|
| `model_version_id` | int | 模型版本 ID |
| `project_id` | int | 项目 ID |
| `run_id` | string | 训练任务 ID |
| `version` | string | 版本号 |
| `stage` | string | 阶段 |
| `description` | string? | 说明 |
| `weights_path` | string? | 权重文件路径 |
| `model_size_mb` | float? | 模型大小 (MB) |
| `metrics` | object? | 模型指标 |
| `created_at` | datetime | 创建时间 |
| `updated_at` | datetime | 更新时间 |
