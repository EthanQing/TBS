# 推理服务 API (Inference)

> **路由前缀：** `/api/v2/inference-runs` 和 `/api/v2/inference-jobs`  
> **标签：** `inference`、`inference-jobs`

推理模块提供基于已注册模型版本的图片/批量/视频推理能力。支持两种模式：
- **轻量推理** (`inference-runs`)：同步执行，适合单次或少量推理
- **推理任务** (`inference-jobs`)：异步执行，支持批量/视频处理和进度跟踪

---

## 一、轻量推理 (`/api/v2/inference-runs`)

### 1. 执行单次推理

```
POST /api/v2/inference-runs
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `model_version_id` | int | ✅ | 模型版本 ID |
| `deployment_id` | int | 否 | 部署 ID |
| `input_path` | string | 二选一 | 本地文件路径或 `token`（上传后获取） |
| `image_url` | string | 二选一 | 图片 HTTP(S) URL |
| `conf` | float | 否 | 置信度阈值（默认 0.5，范围 0~1） |
| `iou` | float | 否 | IOU 阈值（默认 0.45，范围 0~1） |
| `input_meta` | object | 否 | 输入附加信息 |

> ⚠️ `input_path` 和 `image_url` 必须提供其中之一。

**响应：**

```json
{
  "inference_id": 1,
  "model_version_id": 1,
  "input_path": "inference_uploads/abc123.jpg",
  "output": {
    "predictions": [
      { "class_id": 0, "class_name": "person", "confidence": 0.92, "bbox": [100, 200, 300, 500] }
    ],
    "names": { "0": "person", "1": "car" }
  },
  "error_message": null,
  "created_at": "2026-03-20T10:00:00+00:00"
}
```

---

### 2. 上传推理图片

```
POST /api/v2/inference-runs/upload
```

**Content-Type:** `multipart/form-data`

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file` | File | ✅ | 图片或视频文件 |

支持格式：`.jpg`、`.jpeg`、`.png`、`.bmp`、`.webp`、`.tiff`、`.mp4`、`.avi`、`.mov`、`.mkv`、`.wmv`、`.flv`

**响应：**

```json
{
  "token": "inference_uploads/abc123.jpg",
  "path": "/static/temp/inference_uploads/abc123.jpg"
}
```

> 获取的 `token` 可作为推理接口的 `input_path` 参数。

---

### 3. 批量推理

```
POST /api/v2/inference-runs/batch
```

对多张已上传图片依次推理，同步返回。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `model_version_id` | int | ✅ | 模型版本 ID |
| `input_tokens` | string[] | ✅ | 已上传文件的 token 列表 |
| `conf` | float | 否 | 置信度阈值 |
| `iou` | float | 否 | IOU 阈值 |

**响应：**

```json
{
  "results": [
    {
      "filename": "abc123.jpg",
      "token": "inference_uploads/abc123.jpg",
      "output": { "predictions": [...], "names": {...} },
      "error_message": null,
      "inference_time_ms": 15.2
    }
  ],
  "total": 5,
  "success_count": 5,
  "total_time_ms": 123.5
}
```

---

### 4. 视频推理

```
POST /api/v2/inference-runs/video
```

对已上传视频按帧推理。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `model_version_id` | int | ✅ | 模型版本 ID |
| `video_token` | string | ✅ | 视频文件 token |
| `conf` | float | 否 | 置信度阈值 |
| `iou` | float | 否 | IOU 阈值 |
| `frame_interval` | int | 否 | 每隔 N 帧取一帧推理（默认 1） |

**响应：**

```json
{
  "results": [
    { "frame_index": 0, "output": { "predictions": [...] }, "error_message": null },
    { "frame_index": 5, "output": { "predictions": [...] }, "error_message": null }
  ],
  "total_frames": 300,
  "processed_frames": 60,
  "total_time_ms": 5200.0
}
```

## 二、推理任务 (`/api/v2/inference-jobs`)

推理任务是异步执行的推理作业，支持 WebSocket 实时进度推送。

### 1. 获取可推理模型列表

```
GET /api/v2/inference-jobs/models
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `project_id` | int | 按项目过滤 |

返回所有可用于推理的模型列表（包含已注册的模型版本和已完成的训练任务）。

**响应示例：**

```json
[
  {
    "source": "model_version",
    "model_version_id": 1,
    "run_id": "abc-123",
    "project_id": 1,
    "architecture_id": 2,
    "engine": "ultralytics",
    "family": "yolo",
    "variant": "yolov8n",
    "version": "v1.0",
    "label": "yolov8n v1.0 (production)",
    "weights_path": "/path/to/best.pt",
    "inferable": true,
    "created_at": "2026-03-01T12:00:00+00:00"
  }
]
```

---

### 2. 创建推理任务

```
POST /api/v2/inference-jobs
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `mode` | string | 否 | 模式：`image`（默认）/ `batch` / `video` |
| `model_version_id` | int | 二选一 | 模型版本 ID |
| `run_id` | string | 二选一 | 训练任务 ID |
| `input_tokens` | string[] | image/batch 必填 | 已上传文件的 token 列表 |
| `video_token` | string | video 必填 | 视频文件 token |
| `conf` | float | 否 | 置信度阈值（默认 0.5） |
| `iou` | float | 否 | IOU 阈值（默认 0.45） |
| `show_labels` | bool | 否 | 输出显示标签（默认 true） |
| `show_confidence` | bool | 否 | 输出显示置信度（默认 true） |

> ⚠️ `model_version_id` 和 `run_id` 至少提供一个。

**响应：** `201 Created`，返回 `InferenceJobOut` 对象。

---

### 3. 获取推理任务详情

```
GET /api/v2/inference-jobs/{job_id}
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `include_items` | bool | 是否包含推理结果项（默认 true） |

---

### 4. 取消推理任务

```
POST /api/v2/inference-jobs/{job_id}/cancel
```

---

### 5. 推理任务实时流 (WebSocket)

```
ws://<host>/api/v2/inference-jobs/{job_id}/stream?from_seq=0&from_result_id=0
```

#### 消息类型

| type | 说明 |
|------|------|
| `snapshot` | 初始任务快照 |
| `progress` | 进度更新 |
| `item` | 单张图片推理结果 |
| `done` | 任务结束 |
| `error` | 错误 |
| `ping` | 心跳（每 15 秒） |

#### progress 消息示例

```json
{
  "type": "progress",
  "data": {
    "job_id": "job-001",
    "status": "running",
    "phase": "inferring",
    "mode": "batch",
    "progress": 60,
    "processed": 30,
    "total": 50,
    "seq": 15,
    "last_result_id": 30
  }
}
```

#### item 消息示例

```json
{
  "type": "item",
  "data": {
    "result_id": 31,
    "filename": "img_031.jpg",
    "status": "success",
    "detections": 3,
    "inference_time_ms": 12.5,
    "source_url": "/static/temp/inference_uploads/xxx.jpg",
    "output_url": "/static/temp/inference_jobs/job-001/result_031.jpg"
  }
}
```

## 数据模型

### InferenceJobOut

| 字段 | 类型 | 说明 |
|------|------|------|
| `job_id` | string | 任务 ID |
| `status` | string | 状态：`queued`/`running`/`completed`/`failed`/`cancelled` |
| `phase` | string? | 阶段：`preparing`/`inferring`/`rendering`/`finalizing`/`done` |
| `mode` | string | 模式：`image`/`batch`/`video` |
| `progress` | int | 进度百分比 |
| `processed` | int | 已处理数 |
| `total` | int | 总数 |
| `model_version_id` | int? | 模型版本 ID |
| `run_id` | string? | 训练任务 ID |
| `engine` | string? | 推理引擎 |
| `result` | object? | 推理结果 |
| `error_message` | string? | 错误信息 |
| `created_at` | datetime? | 创建时间 |
| `updated_at` | datetime? | 更新时间 |
